# Secure Mail Analyzer

Oltalama (phishing) ve sosyal muhendislik farkindaligi icin gelistirilen,
e-posta icerigi ve link guvenlik analiz platformu. Egitim amaclidir.

## Proje Amaci
Kullanicilarin supheli mail ve linkleri fark etmesini saglayan, risk
seviyesini gerekcesiyle gosteren egitim odakli bir platform.

## Ozellikler
- [ ] E-posta icerigi / link girisi
- [ ] Risk analizi (dusuk / orta / yuksek)
- [ ] Risk gerekcesinin aciklanmasi
- [ ] Analiz gecmisi
- [ ] Admin paneli (istatistikler)
- [ ] Docker ile calistirma
- [ ] Kubernetes deployment

## Kullanilan Teknolojiler
- Backend: .NET Web API (C#)
- Frontend: React
- Veritabani: PostgreSQL
- Container: Docker, Docker Compose
- Orkestrasyon: Kubernetes (Minikube)
